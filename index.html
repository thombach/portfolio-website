🚀 Work

Here is a selection of projects I have worked on. Some of my them are available on my GitHub.

Full stack medical application

Full stack web app for doctors and nurses to help detect malnutrition in nursing homes. It is a school project developed by a team of 6 using agile practices. My roles: frontend, DevOps and scrum master. Partly built with JHipster, the application uses Angular for the frontend and Java Spring for the backend. I used GitHub actions to create CI/CD pipelines, containerized the application with Docker and deployed it on an Azure VM.

Snow and avalanche map

Interactive map updated daily displaying data about snow and avalanches. It provides an overview of the latest weather conditions and thus possible avalanche risks, helping the users of Skitour to prepare for their ski trip in complete safety. The map has been developed with Leaflet (frontend) and PHP (backend), by a team of 4 using agile practices.

Electronic cards test manager

Complete user interface I made with Angular, ElectronJS and MySQL during a 3 month-internship, to manage tests of electronic cards that are manufactured by the company. With this software, the user can launch test scripts on a card with a hardware bench, consult previous test history with all the stored data, and visualize graphs and statistics in order to detect potential problems and improve the engineering process.

Experimental project using AI applied to sound recognition on an embedded device. That was my 4th year end project completed in a few weeks by a team of 3.
